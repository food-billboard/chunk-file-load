# chunk-file-load 

## 介绍 
文件分片上传工具  
支持`Blob`、`File`、`ArrayBuffer`、`Base64`类型的文件上传  
支持断点续传
支持控制上传  

## Demo  
- 实际文件上传`Demo`  
`fork`[github](https://github.com/food-billboard/chunk-file-load)代码  
本地执行`npm run start`和`npm run server` 执行本地调试  
- `mock`后台`Demo`  
访问[Demo]()地址
